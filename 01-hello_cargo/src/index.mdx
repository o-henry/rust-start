## Cargo

Cargo is Rust’s build system and package manager.

- cargo new "project_name" --bin
- cargo check : 실행 파일 생성을 생략하므로 build 명령보다 훨씬 빠르게 실행된다. 따라서 코드가 작업결과(컴파일)에 오류가 없는지 확인하려면 check 명령을 수시로 이용한다.
- cargo build
- cargo run
- cargo build --release
